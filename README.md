##项目介绍

###1.采用的技术
前端实现，采用html+js+css的结构，html中加入echarts框架，使用Ajax获取疫情实时数据。
###2.实现的功能
可以选择日期，通过地图颜色的深浅展示疫情的严重程度。
实现了地图二级联动，可以查看具体省份。
疫情趋势图，包含全国累计、现存、新增趋势，可以选择性查看确诊、死亡、治愈三种折线图。
